## Grade Clicker

### 1. Название проекта
**Grade Clicker** — интерактивное приложение для набора баллов. Пользователь кликает по изображению, получая баллы. При достижении определённого количества баллов изменяется картинка, показывающая баллы.

### 2. Цель работы
Изучить жизненный цикл Activity (Activity Lifecycle) в Android и научиться правильно обрабатывать переходы между состояниями. В результате работы создано приложение, демонстрирующее работу lifecycle callbacks и сохранение состояния при изменении конфигурации устройства.

### 3. Ответьте на контрольные вопросы

- **Какие три метода вызываются при запуске приложения?**  
  При запуске последовательно вызываются: `onCreate()`, `onStart()`, `onResume()`.

- **Чем отличается onPause() от onStop()?**  
  `onPause()` вызывается, когда Activity теряет фокус (частично видна). `onStop()` вызывается, когда Activity становится полностью невидимой для пользователя (находится в фоне).

- **Когда вызывается onRestart()?**  
  Метод `onRestart()` вызывается только тогда, когда Activity перезапускается после того, как была остановлена (после вызова `onStop()`), например, при возвращении приложения из фона.

- **Что происходит при повороте экрана?**  
  Происходит изменение конфигурации устройства. Android полностью уничтожает текущую Activity (вызывается `onDestroy()`), затем создаёт новую (вызывается `onCreate()`). Все данные теряются, если они не были сохранены.

- **В чём разница между remember и rememberSaveable?**  
  `remember` сохраняет значение только во время рекомпозиции, но теряет данные при пересоздании Activity (поворот экрана). `rememberSaveable` сохраняет значение как при рекомпозиции, так и при изменении конфигурации устройства.

- **Почему onCreate() должен вызывать super.onCreate()?**  
  Вызов обязателен, так как родительский класс Activity выполняет критически важную инициализацию. Без этого система Android не сможет правильно создать Activity и обеспечить работу жизненного цикла.

- **Для чего используется класс Log?**  
  Класс `Log` используется для отладки и мониторинга приложения. Он записывает сообщения в системный лог (Logcat), позволяя отслеживать выполнение кода и переходы между состояниями.

- **Какой метод вызывается только один раз за время жизни Activity?**  
  Метод `onCreate()` вызывается только один раз за весь жизненный цикл Activity — в момент её создания.
